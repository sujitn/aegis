# Extension Icons

Add the following PNG icons to this directory:

## Required Icons
- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Offline State Icons (optional)
- `icon16-offline.png` - 16x16 pixels (grayed/red)
- `icon32-offline.png` - 32x32 pixels (grayed/red)
- `icon48-offline.png` - 48x48 pixels (grayed/red)
- `icon128-offline.png` - 128x128 pixels (grayed/red)

## Design Guidelines
- Use a shield icon representing protection
- Primary color: #1a73e8 (blue)
- Offline color: #9e9e9e (gray) or #f44336 (red)
- Keep design simple and recognizable at small sizes
